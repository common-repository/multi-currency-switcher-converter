== Installation ==

1 - Settings > Currency Switcher

2 - Get a FREE API key on https://bankersalgo.com, then initialize the rates

3 - Add the following shortcodes in your Posts, Pages or Custom HTML widgets / or PHP code anywhere in your theme :

Switcher:

Shortcode example: [switcher currencies="USD,AUD,CAD,CHF,EUR,GBP,CRYPTO_BTC,CRYPTO_ETH" type=1]
PHP example: html_switcher( 'USD,AUD,CAD,CHF,EUR,GBP,CRYPTO_BTC,CRYPTO_ETH', 1 );
- currencies: currencies to add in the switcher, first one should be your main currency
- type: currency name format (0: code - currency name - symbol) (1: code) (2: code symbol) (3: currency name - symbol) 

Prices:

Shortcode example: [price number=99.90 currency="USD"] 
PHP example: echo get_price( 'number=99.90&currency=USD' );
- number: price 
- currency: initial currency code 

Symbols:

Shortcode example: [symbol currency="USD"]
PHP example: echo get_symbol( 'currency=USD' );
- currency: initial currency code
